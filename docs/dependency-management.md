# Закрепление зависимостей

Production-зависимости backend закреплены в `backend/requirements.lock`. Файл содержит
версию каждого прямого пакета приложения и критичных транзитивных пакетов. Обновление
выполняется отдельным изменением с прогоном unit-тестов и production build.

Для воспроизводимой установки используйте:

```bash
python -m pip install --requirement backend/requirements.lock
```

Локальная среда разработки может использовать `pip install -e ".[dev]"`, но перед
релизом версии должны быть синхронизированы с lock-файлом.
